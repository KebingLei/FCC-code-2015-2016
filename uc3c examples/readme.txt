This zip file provides a set of basic drivers for UC3C for GCC and IAR.

GCC Installer can be found there:
http://www.atmel.no/beta_ware/


IAR Installer can be found there:
http://www.iar.com/website1/1.0.1.0/3/1/


Applying the patch: 
Unzip the avr32-headers.zip file in the IAR EWAVR32 installation folder under
"IAR installation folder"\Embedded Workbench 5.3\avr32\inc or in the GNU Toolchain 
installation folder under C:\Program Files\Atmel\AVR Tools\AVR32 Toolchain\avr32\include\avr32 (default location).
